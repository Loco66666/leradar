# Le Radar

Bot Discord professionnel orienté veille financière, pédagogie et analyse.

## Stack
Node.js, TypeScript, Discord.js, Prisma, SQLite (MVP), zod, dotenv, pino, Docker.

## Installation
1. `npm install`
2. Copier `.env.example` vers `.env`
3. `npm run prisma:migrate`
4. `npm run dev`

## Scripts
- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`
- `npm run prisma:migrate`
- `npm run prisma:studio`

## Commandes V1
- `/help`
- `/price asset`
- `/fear-greed`
- `/market-pulse`
- `/news sujet`
- `/analyse question`

## Disclaimer
Information éducative, pas un conseil financier.
